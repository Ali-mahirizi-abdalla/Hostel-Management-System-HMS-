## Project Overview
This Hostel Management System is a comprehensive web-based application designed to streamline hostel operations, including:
- Student accommodation management
- Meal booking and tracking
- Maintenance request handling
- Leave request management
- M-Pesa payment integration
- Real-time analytics and reporting

## Technology Stack
- **Backend:** Django 4.2 (Python)
- **Frontend:** HTML5, Tailwind CSS, JavaScript
- **Database:** SQLite (Development) / PostgreSQL (Production)
- **Payment Gateway:** M-Pesa Daraja API
- **Authentication:** Django Auth + Google OAuth
- **Deployment:** Render, Vercel compatible

## Key Features
✅ Student and Admin dashboards  
✅ Real-time meal booking system  
✅ Away mode for automatic meal cancellation  
✅ Maintenance request tracking with photo uploads  
✅ Leave request approval workflow  
✅ M-Pesa STK Push payment integration  
✅ Analytics and reporting  
✅ Dark mode support  
✅ Responsive design  
✅ Email notifications
